# i200 Mäng "Frogger"

i200 kursuse jaoks tehtava mängu "Frogger" repositoorium Mängu idee tuleneb ning teostus loodetavasti sarnaneb 90 date algusest pärineva mänguga "Frogger". Pilte ning infot leiab siit: https://en.wikipedia.org/wiki/Frogger Projekti eesmärk on selgeks õppida Java algtõed ning neid praktiseerides ka enda arukust tõsta.
